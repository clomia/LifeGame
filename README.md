# 콘웨이의 Life Game을 3차원으로 구현한 게임 만들기
------------
> #### 목표
> * 2D환경에서 기초적으로 구현해보기 [ursina 패키지 이해를 위해서]
>   * 1 기초적인 수준 - 세포를 배치하고 관찰하기
>   * 2 세포를 두 새력으로 나누어서 한쪽만 살아남는 규칙을 구상해서 게임으로 구현하기
> * #### 3D 환경으로 구현하기
>   * 1 정육면체를 세포로 하여서 X축,Y축(평면)으로만 구현해보기
>   * 2 라이프 게임을 3차원 환경에서 구현하기 위한 규칙 구상하기
>   * 3 (2)에서 고안한 규칙으로 3차원 환경에서 라이프 게임을 구현하기
>   * 4  3차원 환경에서 여러 새력으로 나누어진 세포들의 변화를 시뮬레이션하도록 만들기
>   * 5  단일 컴퓨터로 사용하는 게임으로 완성해서 실행파일 만들기
> * 온라인 게임으로 만들기
>   * __보류__
